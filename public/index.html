NJW Xarvis

AI Assistant Extension for Your Browser

A cross-browser AI sidepanel extension for Chrome, Firefox, and Brave. Chat with AI using live page context without leaving your tab.

✨ Key Features

Context-Aware Chat

Send URL, text, elements, or screenshots with every message

Multi-Provider

Switch between Gemini, Claude, ChatGPT, and more

Session Memory

Keep history, resume chats, and track context

Keyboard Shortcut

Quick launch with Cmd+Shift+Y (macOS)
